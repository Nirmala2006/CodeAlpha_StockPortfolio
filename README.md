# Stock Portfolio Tracker

## 📌 Project Description
This is a simple Python project that calculates the total investment value based on user input stock names and quantities.

The program uses a dictionary to store stock prices and allows users to input multiple stocks to calculate the final portfolio value.

---

## 🚀 Features
- Add multiple stocks
- Calculate total investment value
- Simple user input system
- Handles invalid stock names

---

## 🧠 Concepts Used
- Dictionary
- Loops (while loop)
- Conditional Statements (if-else)
- Input / Output
- Arithmetic Operations

---

## 💻 How it works
1. User enters stock name
2. User enters quantity
3. Program calculates total value
4. Stops when user types "done"

---

## 📦 Example Stocks Used
- AAPL → 150
- TSLA → 250
- GOOGLE → 2800
- AMAZON → 3300
